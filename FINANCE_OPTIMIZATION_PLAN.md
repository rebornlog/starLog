#### 1. 基金数据接入 ⏱️ 4h

**实施内容：**
```python
# services/finance/funds.py
import akshare as ak

# 获取基金实时数据
def get_fund_realtime_data(fund_type='混合基金'):
    """
    获取基金实时数据
    类型：混合基金/股票基金/债券基金/指数基金/QDII
    """
    fund_df = ak.fund_open_fund_info_em(fund=fund_type, indicator="单位净值")
    return fund_df.to_dict('records')

# 获取基金持仓
def get_fund_holdings(fund_code):
    """获取基金前十大持仓"""
    holdings = ak.fund_portfolio_hold_em(symbol=fund_code)
    return holdings.to_dict('records')
```

**API 端点：**
```
GET /api/funds - 获取基金列表
GET /api/funds/[code] - 获取基金详情
GET /api/funds/[code]/holdings - 获取基金持仓
```
