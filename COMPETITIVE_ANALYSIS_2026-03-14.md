## 🎨 设计规范建议

### 色彩系统
```
主色：#10B981 (Emerald 500) - 当前使用 ✅
辅色：#3B82F6 (Blue 500) - 链接/按钮
成功：#10B981 - 涨/正面
警告：#F59E0B - 注意/提醒
危险：#EF4444 - 跌/错误
```

### 字体大小
```
标题：2xl-4xl (24px-36px)
正文：base-lg (16px-18px)
辅助：sm-xs (14px-12px)
```

### 间距规范
```
小间距：p-2/p-3 (8px/12px)
中间距：p-4/p-6 (16px/24px)
大间距：p-8/p-12 (32px/48px)
```
