### 3.2 相关文章推荐 ✅

**功能：** 文章详情页底部推荐相关文章

**推荐算法：**
1. **筛选条件：**
   - 同分类（category 匹配）
   - 同标签（tags 有交集）
   - 排除当前文章
2. **排序规则：**
   - 按阅读量降序（viewCount DESC）
3. **数量限制：**
   - 最多 3 篇

**Prisma 查询：**
```typescript
const relatedPosts = await prisma.post.findMany({
  where: {
    isPublished: true,
    id: { not: currentPost.id },
    OR: [
      { category: currentPost.category }, // 同分类
      { tags: { hasSome: currentPost.tags } }, // 同标签
    ],
  },
  take: 3,
  orderBy: { viewCount: 'desc' },
})
```

**UI 设计：**
```tsx
<section className="mt-12 pt-8 border-t">
  <h2>📖 相关文章</h2>
  <div className="grid md:grid-cols-3 gap-4">
    {relatedPosts.map(post => (
      <ArticleCard post={post} />
    ))}
  </div>
</section>
```

**卡片内容：**
- 文章标题（最多 2 行）
- 摘要（最多 2 行）
- 发布日期
- 阅读量

**交互效果：**
- Hover 阴影增强
- 向上浮动（-translate-y-1）
- 标题颜色变化（emerald 绿）

**代码变更：**
- 文件：`app/blog/[slug]/page.tsx`
- 优化查询：OR 条件 + 阅读量排序
- 新增 UI：3 列网格卡片
