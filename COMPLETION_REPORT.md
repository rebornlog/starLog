### 2. K 线数据接口（P0）⭐新增
**接口：** `GET /api/stocks/{code}/kline`

**功能：**
- ✅ 支持日 K/周 K/月 K（period=day/week/month）
- ✅ 数据源：新浪财经 API
- ✅ 备用方案：模拟数据（当新浪接口失败时）
- ✅ 5 分钟缓存
- ✅ 返回 100 根 K 线数据

**测试：**
```bash
curl http://localhost:8081/api/stocks/000001/kline?period=day
# ✅ 返回 100 条 K 线数据
```
