### 建议 2：评论系统启用

**配置 Giscus：**

```bash
# 1. 创建 GitHub Discussion
# 访问：https://github.com/rebornlog/starLog/discussions

# 2. 安装 Giscus App
# 访问：https://github.com/apps/giscus

# 3. 获取配置
# 访问：https://giscus.app/zh-CN

# 4. 更新配置
# 编辑：data/siteMetadata.js
comments: {
  provider: 'giscus',
  giscusConfig: {
    repo: 'rebornlog/starLog',
    repositoryId: 'R_kgDOxxxxxx',  // 从 giscus.app 获取
    category: 'Announcements',
    categoryId: 'DIC_kwDOxxxxxx',   // 从 giscus.app 获取
  }
}
```

---

### 建议 3：Umami 统计启用

**步骤：**

1. **注册 Umami Cloud**
   - 访问：https://cloud.umami.is/
   - 免费注册

2. **添加网站**
   - 网站名称：starLog
   - 域名：starlog.dev

3. **获取 Website ID**
   - 从后台复制 ID

4. **更新配置**
   ```js
   // data/siteMetadata.js
   analytics: {
     umamiAnalytics: {
       umamiWebsiteId: 'your-website-id',
     },
   }
   ```

5. **重启服务**
   ```bash
   pm2 restart starlog-frontend
   ```

---

### 建议 4：CDN 配置

**Cloudflare 免费套餐：**

1. **注册账号**
   - 访问：https://dash.cloudflare.com/sign-up

2. **添加域名**
   - starlog.dev

3. **修改 DNS**
   - 使用 Cloudflare DNS 服务器

4. **配置缓存规则**
   - 静态资源缓存 1 个月
   - API 不缓存

5. **开启优化**
   - Auto Minify
   - Brotli 压缩
   - HTTP/2

**预期效果：** 加载速度提升 50%+

### 建议 5：SEO 结构化数据

**实施 Schema.org：**

```tsx
// components/seo/StructuredData.tsx
import Script from 'next/script'

export function ArticleStructuredData({ post }) {
  return (
    <Script
      type="application/ld+json"
      dangerouslySetInnerHTML={{
        __html: JSON.stringify({
          '@context': 'https://schema.org',
          '@type': 'BlogPosting',
          headline: post.title,
          datePublished: post.date,
          dateModified: post.lastmod,
          author: {
            '@type': 'Person',
            name: '老柱子',
          },
        }),
      }}
    />
  )
}
```
