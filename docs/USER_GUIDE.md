# starLog 用户使用手册

## 📱 功能概览

starLog 是一个专业的基金投资管理平台，提供实时基金数据、自选管理、价格提醒、定投计算等功能。

---

## 🎯 核心功能

### 1. 基金列表 📊

**访问路径：** `/funds`

**功能特性：**
- 🔍 **搜索**：支持基金代码/名称模糊搜索
- 🏷️ **类型筛选**：股票型、混合型、债券型、货币型、QDII、指数型
- ⚠️ **风险等级**：低、中低、中、中高、高
- 📊 **排序**：按代码、净值、涨跌幅排序
- 🔄 **刷新**：手动刷新或自动刷新（45 分钟）
- 📈 **对比**：选择多只基金进行对比

**使用技巧：**
```
1. 快速搜索：输入基金代码或名称关键字
2. 多选对比：点击"基金对比"按钮，选择最多 5 只基金
3. 查看净值：绿色表示今天数据，橙色表示历史数据
```

---

### 2. 基金详情 📈

**访问路径：** `/funds/{基金代码}`（如 `/funds/005827`）

**功能特性：**
- 📊 **基本信息**：净值、涨跌、类型、规模、基金经理
- 📉 **历史业绩**：历史净值走势图表
- 💼 **持仓分布**：资产配置、前十大持仓、行业分布
- ⭐ **加入自选**：一键添加到自选列表
- 📊 **基金对比**：直接发起对比

**选项卡说明：**
- **基本信息**：基金档案、基金经理、费率结构
- **历史业绩**：净值走势、历史数据表格
- **持仓分布**：资产配置、股票持仓、行业分布

---

### 3. 自选管理 ⭐

**访问路径：** `/funds/watchlist`

**功能特性：**
- ➕ **添加自选**：从详情页或列表页添加
- 🗑️ **移除自选**：单个移除或清空全部
- 📊 **实时数据**：自动获取最新净值
- 💾 **持久化**：数据存储在本地浏览器

**使用技巧：**
```
1. 快速添加：在基金列表点击"⭐ 加入自选"
2. 批量管理：在自选页可以清空所有
3. 数据同步：使用同一浏览器自动同步
```

---

### 4. 价格提醒 🔔

**访问路径：** `/funds/alerts`

**功能特性：**
- 📈 **高于目标价**：净值上涨突破时提醒
- 📉 **低于目标价**：净值下跌突破时提醒
- ⏰ **触发记录**：记录触发时间
- 🔕 **启用/禁用**：灵活控制提醒状态

**使用步骤：**
```
1. 点击"添加价格提醒"
2. 输入基金代码（如 005827）
3. 选择提醒类型（高于/低于）
4. 设置目标价格（如 2.00）
5. 确认添加
```

**提醒规则：**
- 高于目标价：当前净值 ≥ 目标价格时触发
- 低于目标价：当前净值 ≤ 目标价格时触发
- 触发后显示红色高亮，记录触发时间

---

### 5. 基金对比 📊

**访问路径：** `/funds/compare`

**功能特性：**
- 🔗 **URL 分享**：支持 `?codes=xxx,yyy` 参数分享
- 📈 **关键指标**：净值、涨跌、类型、更新时间
- 📉 **业绩走势**：近 30 日涨幅对比
- 📊 **最多 5 只**：一次对比最多 5 只基金

**使用技巧：**
```
1. 快速对比：/funds/compare?codes=005827,003096
2. 从列表选择：在基金列表选择多只后点击"开始对比"
3. 分享链接：复制 URL 发送给好友
```

---

### 6. 定投计算器 💰

**访问路径：** `/funds/sip-calculator`

**功能特性：**
- 💵 **每月定投**：100 元起，支持自定义
- 📅 **定投年限**：1-30 年
- 📈 **预期收益率**：参考不同类型基金收益率
- 📊 **收益走势**：可视化展示本金和收益
- 📋 **详细数据**：按年展示投资收益

**参考收益率：**
```
货币基金：2-3%
债券基金：4-6%
混合基金：8-12%
股票基金：10-15%
```

**计算示例：**
```
每月定投：1000 元
定投年限：3 年
预期年化：8%

结果：
- 总投入：36,000 元
- 总资产：39,526 元
- 累计收益：3,526 元
- 收益率：+9.79%
```

### 7. 导入导出 📥📤

**导入路径：** `/funds/import`  
**导出路径：** `/funds/export`

**导入格式：**
```
支持格式：
- 仅代码：005827
- 代码 + 名称：005827,易方达蓝筹
- 代码 + 名称 + 备注：005827,易方达蓝筹，张坤管理

分隔符：逗号、制表符、空格
```

**导出格式：**
- CSV 格式
- Excel 格式（可选）
- 包含字段：代码、名称、净值、涨跌、类型

## ⚠️ 注意事项

### 数据更新

- **更新时间：** 交易日 15:00 后更新当日净值
- **QDII 基金：** 因时差问题，净值延迟 1-2 天
- **货币基金：** 净值固定为 1.0000，收益体现在份额

### 风险提示

```
⚠️ 基金有风险，投资需谨慎

1. 过往业绩不代表未来表现
2. 净值波动可能导致本金损失
3. 请根据自身风险承受能力选择
4. 定投计算器仅供参考，不构成投资建议
```

### 数据说明

- **单位净值：** 每份基金的当前价值
- **累计净值：** 单位净值 + 累计分红
- **日涨跌幅：** 相对于上一交易日的涨跌幅
- **更新时间：** 数据来源时间戳

## 🆘 常见问题

### Q1: 数据不更新怎么办？

**A:** 
1. 点击页面"🔄 刷新"按钮
2. 检查网络连接
3. 等待后台自动刷新（45 分钟）

### Q2: 自选数据丢失？

**A:** 
- 自选数据存储在本地浏览器
- 清除缓存会丢失数据
- 建议使用导入功能备份

### Q3: 价格提醒不触发？

**A:** 
1. 检查提醒是否启用
2. 确认目标价格设置正确
3. 刷新页面更新最新净值

### Q4: 对比页无法打开？

**A:** 
1. 检查 URL 参数格式
2. 确保基金代码正确
3. 最多对比 5 只基金
