## 产品介绍

**starLog** 是一款现代化的个人知识库系统，支持：

- 📝 **技术博客** - Markdown 写作，代码高亮
- 📊 **金融分析** - 数据可视化，图表展示
- 🗂️ **项目管理** - 项目文档，进度跟踪
- 🎨 **多主题** - 10 套精美主题，随心切换
- 📱 **响应式** - 完美适配手机、平板、电脑

### 核心特性

| 特性 | 说明 |
|------|------|
| 多主题系统 | 10 套高级感主题，一键切换 |
| Markdown 支持 | 完整 Markdown + 数学公式 + 代码高亮 |
| SEO 优化 | 自动生成 sitemap、RSS 订阅 |
| 暗色模式 | 支持明暗主题切换 |
| 快速搜索 | 全文搜索，快速定位 |
| 响应式设计 | 适配各种屏幕尺寸 |

## 主题切换

### 如何切换主题

1. **找到主题按钮** - 页面右上角 🎨 图标
2. **点击打开面板** - 显示所有可用主题
3. **选择喜欢的主题** - 点击即可应用
4. **自动保存** - 下次访问保持选择

## 内容管理

### 阅读文章

1. 进入 **博客** 页面
2. 浏览文章列表

1. 点击搜索图标 🔍
2. 输入关键词

4. 快速定位内容

### 标签浏览

1. 进入 **标签** 页面
2. 查看所有标签
3. 点击标签筛选文章

## 技术支持

### 遇到问题？

#### 1. 页面加载慢
- 清除浏览器缓存 (Ctrl+Shift+R)
- 检查网络连接
- 尝试无痕模式

#### 2. 主题不显示
- 刷新页面
- 清除 localStorage
- 检查浏览器兼容性

#### 3. 搜索不工作
- 等待页面完全加载
- 检查 JavaScript 是否启用
- 刷新页面重试

## 开发者说明

### 本地开发

```bash
git clone https://github.com/rebornlog/starLog.git
cd starLog
npm install
npm run dev
```

### 添加新文章

在 `content/blog/` 目录创建 Markdown 文件：

```markdown
---
title: 文章标题
date: 2026-03-05
tags: [标签 1, 标签 2]
summary: 文章摘要
---

文章内容...
```

### 添加新项目

在 `content/projects/` 目录创建 Markdown 文件。

### 部署更新

```bash
git add .
git commit -m "更新内容"
git push
```
