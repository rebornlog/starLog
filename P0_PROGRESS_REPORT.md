## ⚠️ 待确认问题

### 1. 基金数据为 0 问题
**现状：**
- 部分基金（如 161725 招商中证白酒）净值为 0
- API 返回数据正常，但部分基金确实无实时数据

**原因：**
- 天天基金网部分基金暂停申购/赎回
- 或 QDII 基金因时差问题数据延迟

**建议：**
- 添加数据更新时间显示
- 对于无数据基金显示"暂停交易"标签
- 或切换到多数据源（新浪财经备用）
